# Othello🎮🧠
**オセロゲーム with com 対戦機能**  
プレイヤー vs COM で遊べるオセロゲーム！  
ブラウザで動作し、シンプルな操作で対戦が可能。

## 🎯 **仕様**
人間は先攻。comは後攻。
comはその時の盤でもっともひっくり返せる石が多い場所に打ってきます。

## 📖 **遊び方**
1. **プレイヤーの手番:** クリックで駒を置く  
2. **COMの手番:** 自動で最適な手を選んで打ってくる 
3. **試合終了:** どちらも置けなくなったら終了  

🔄 **「戻る」「進む」ボタンで手を巻き戻し / 進めることも可能！**
